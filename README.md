- Add user reviews for books
- Responsive design for different screen sizes
- Simple navigation and clean UI layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Purpose
This project demonstrates **frontend development skills** including responsive design, DOM manipulation, and building interactive user interfaces. Ideal for showcasing skills as a **fresher frontend developer**.
